Congratulations on successfully installing and configuring MineTrax. You can now:

## Read the documentation

Continue reading the documentation to learn more about MineTrax and its features.
Experiment with the different features and see what you can do with MineTrax.
Not everything is covered in the documentation yet (but we are working on it). So you might find features that are not documented yet.

## Give review on SpigotMC

If you like MineTrax, please consider giving a review on [SpigotMC](https://www.spigotmc.org/resources/minetrax-suite.102378/).

MineTrax is a free and open-source project & will always remain one. Giving a review is a great way to show your appreciation and support the project.

## Join Discord

Join the [Discord server](https://discord.gg/Hzfj27k) to get support, discuss features, and get the latest updates.

## Contribute to the project

MineTrax is a free and open-source project. Any kind of contribution is welcome. You can contribute to the project in many ways:

- Report bugs and suggest features.
- Help other users on Discord.
- Improve the documentation.
- Improve the code.
- Create a pull request to add new features.
- Spread the word about MineTrax.
